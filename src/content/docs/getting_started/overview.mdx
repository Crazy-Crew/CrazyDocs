## Contributing
Contributing to CrazyDocs is the easiest thing in the world!
<Steps>
1. git clone https://github.com/ryderbelserion/Crux.git
2. pnpm install
3. pnpm dev
4. Edit files in `content/docs`, It will refresh immediately with new changes
</Steps>
