**This page lists all permissions included in CrazyEnvoys.**

## Commands
| Command      | Aliases |
|--------------|---------|
| /crazyenvoys | /cev    |

### Auction Permissions
| Command                          | Permission Node  | Description                                                                    | Player has by default? |
|----------------------------------|------------------|--------------------------------------------------------------------------------|------------------------|
| /envoy /envoy time               | envoy.time       | Shows the time till the envoy starts or ends.                                  | Yes                    |
| /envoy help                      | envoy.help       | Shows all crazy envoy commands.                                                | Yes                    |
| /envoy reload                    | envoy.reload     | Reloads all the files and tiers.                                               | No                     |
| `/envoy drops [page]`            | envoy.drops      | Shows all the current crate locations.                                         | Yes                    |
| /envoy ignore                    | envoy.ignore     | Mutes the envoy collecting message for the user.                               | Yes                    |
| `/envoy flare [amount] [player]` | envoy.flare.give | Gives a player a flare to start the envoy.                                     | No                     |
| /envoy edit                      | envoy.edit       | Sets the player into editor mode which allows them to edit the drop locations. | false                  |
| /envoy start                     | envoy.start      | Starts the envoy event.                                                        | false                  |
| /envoy stop                      | envoy.stop       | Stops the current envoy event.                                                 | false                  |
| /envoy center                    | envoy.center     | Sets the center for the random crate drops.                                    | false                  |

### Misc Permissions
| Permission Node | Description                                             | Player has by default? |
|-----------------|---------------------------------------------------------|------------------------|
| envoy.flare.use | Allows you to use flares.                               | No                     |
| envoy.locations | Allows you to see envoy locations when an event starts. | No                     |
| envoy.bypass    | Allows you to use all commands.                         | No                     |
