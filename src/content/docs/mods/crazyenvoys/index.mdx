### Current Features
- In-game location editing with multi-user support.
- Unlimited locations & prizes.
- Configurable warning times.
- Configure the falling block & crate block.
- Ability to configure the firework colors.
- Customize almost every message.
- Signal Flares, Flares, Multiple tiers.
- Ability to have envoy claim cooldowns.
- Ability to have a grace period for envoys to prevent people rushing it.
- and so much more!
