    <TabCard label="ItemsAdder">
        <Aside type="tip">
            All identifiers for items can be found via `ItemsAdder/contents` folder

            ```yml
            info:
              namespace: iaalchemy -> this is the namespace.
            items:
              arcane_ring: -> arcane_ring is the identifier.
                enabled: true
                display_name: display-name-arcane_ring
                permission: iaalchemy.items.arcane_ring
                lore:
                  - lore-1-arcane_ring
                  - lore-2-arcane_ring
                resource:
                  material: TOTEM_OF_UNDYING
                  generate: true
                  textures:
                    - item/arcane_ring
               attribute_modifiers:
                 mainhand:
                   movementSpeed:
                     value: 0.1
                     operation: multiply_base
                 offhand:
                   movementSpeed:
                     value: 0.1
                     operation: multiply_base
            ```

            iaalchemy:arcane_ring is the combined result needed!
        </Aside>
		
		`iaalchemy:arcane_ring` can be used in any section that asks for a material such as `diamond_axe`.

        ### Physical Keys
        ```yml
        PhysicalKey:
          # Name of the Key.
          Name: "A fancy key using Nexo."
          # Lore of the Key.
          Lore: []
          # The custom item from Nexo.
          Item: "iaalchemy:arcane_ring"
        ```
    </TabCard>

    <TabCard label="Nexo">
        <Aside type="tip">
            All identifiers for items can be found via `/nexo give [item]`, or `nexo_defaults/items` folder

            ```yml
            forest_axe: -> forest_axe is the identifier.
              itemname: Forest Axe
              material: NETHERITE_AXE
              Pack:
                model: nexo:item/nexo_tools/forest_axe
                custom_model_data: 1000
            ```
        </Aside>

        `forest_axe` can be used in any section that asks for a material such as `diamond_axe`.

        ### Physical Keys
        ```yml
        PhysicalKey:
          # Name of the Key.
          Name: "A fancy key using Nexo."
          # Lore of the Key.
          Lore: []
          # The custom item from Nexo.
          Item: "forest_axe"
        ```
    </TabCard>
